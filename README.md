# Web Design Tool

Local-only AI design tool for generating and iterating on website designs.

## Setup

1. `cp .env.example .env` and add your `ANTHROPIC_API_KEY` (get one at https://console.anthropic.com/settings/keys)
2. `npm run install:all`
3. `npm start`

Frontend: http://localhost:5173 — Backend: http://localhost:3001

Projects are stored in `/projects/` (gitignored).
